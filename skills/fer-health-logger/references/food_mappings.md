# Food Mappings

Use this reference when translating Max's Spanish or casual food phrases into USDA FoodData Central queries.

## Defaults

| User phrase | Log item | USDA query | Notes |
| --- | --- | --- | --- |
| huevos, eggs | eggs | egg whole cooked hard-boiled | Plain cooked default. Ask if fried, scrambled with oil/butter, or cheese matters. |
| frijoles, beans | beans | black beans cooked | Use as generic frijoles only when type is not specified. If Max says bayo/pinto/refried, use that instead. |
| cafe, café, coffee | coffee | coffee brewed | Black brewed coffee default. Ask if milk, sugar, creamer, or latte is mentioned. |

## Rules

- Prefer a plain preparation default when the user does not specify cooking fat or extras.
- Add preparation notes when a default is used.
- If preparation materially changes calories and cannot be safely defaulted, ask one short question before logging.
- Keep one row per food or beverage item.
